# 01_Vision.md

# WorkspaceGraph

**Version:** 0.1 (Draft)

------------------------------------------------------------------------

# Vision

WorkspaceGraph adalah sebuah **AI Workspace Operating System** yang
menjadikan seluruh pengetahuan, pekerjaan, proyek, aturan, dan
pengalaman pengguna sebagai sebuah sistem yang hidup, saling terhubung,
dapat dipahami oleh manusia maupun AI, serta tetap dimiliki sepenuhnya
oleh pengguna.

WorkspaceGraph bukan sekadar aplikasi pencatat, bukan sekadar AI Chat,
dan bukan sekadar Graph Viewer.

WorkspaceGraph adalah lingkungan kerja terpadu yang menggabungkan:

-   Knowledge Management
-   AI Assistant
-   Project Management
-   Task Management
-   Graph Knowledge
-   Markdown Workspace

ke dalam satu ekosistem.

------------------------------------------------------------------------

# Mission

Membantu seseorang membangun "Second Brain" yang dapat:

-   berpikir bersama AI,
-   berkembang setiap hari,
-   menyimpan seluruh pengetahuan,
-   menghubungkan seluruh informasi,
-   dan tetap dapat digunakan bahkan tanpa bergantung pada model AI
    tertentu.

------------------------------------------------------------------------

# Philosophy

WorkspaceGraph dibangun berdasarkan keyakinan bahwa:

> AI bukan pemilik pengetahuan.

AI hanyalah pekerja.

Pengetahuan tetap dimiliki pengguna.

Seluruh pengetahuan harus berada pada file yang dapat dibaca manusia.

------------------------------------------------------------------------

# Core Philosophy

## Everything is Markdown.

Semua informasi utama disimpan sebagai file Markdown.

Bukan database.

Bukan format proprietary.

## Everything is Connected.

Tidak ada catatan yang berdiri sendiri.

Semua catatan dapat saling terhubung.

## Everything has Context.

AI tidak boleh menjawab tanpa memahami konteks.

Sebelum bekerja AI harus memahami:

-   project
-   template
-   SOP
-   knowledge
-   task
-   history
-   relationship

yang relevan.

## AI is a Worker.

AI bukan pusat sistem.

AI hanyalah mesin yang membantu mengerjakan pekerjaan.

WorkspaceGraph tetap dapat digunakan tanpa AI.

## User Owns Everything.

Seluruh data dimiliki pengguna.

Pengguna bebas:

-   membuka file
-   memindahkan file
-   backup
-   menggunakan Git
-   sinkronisasi cloud
-   membuka dengan editor lain

tanpa kehilangan data.

------------------------------------------------------------------------

# Product Vision

WorkspaceGraph ingin menjadi tempat dimana seluruh aktivitas pengguna
berada.

Misalnya:

-   belajar
-   bekerja
-   membuat proyek
-   membuat SOP
-   menulis jurnal
-   membuat dokumentasi
-   menyimpan template
-   mengelola tugas
-   mengelola pengetahuan

Semuanya berada dalam satu Workspace.

------------------------------------------------------------------------

# Long-Term Vision

WorkspaceGraph diharapkan berkembang menjadi:

> Operating System untuk Knowledge Worker.

Seorang pengguna cukup membuka WorkspaceGraph untuk memulai seluruh
aktivitasnya.

Tidak perlu berpindah-pindah aplikasi hanya untuk mengelola pengetahuan.

------------------------------------------------------------------------

# AI Vision

WorkspaceGraph tidak dibangun untuk satu AI tertentu.

Hari ini mungkin menggunakan Gemini CLI.

Besok dapat menggunakan:

-   Claude
-   GPT
-   Grok
-   Qwen
-   DeepSeek
-   Model lokal

AI hanyalah Engine.

WorkspaceGraph adalah sistemnya.

------------------------------------------------------------------------

# Knowledge Vision

Knowledge bukan kumpulan file.

Knowledge adalah Graph.

Hubungan antar informasi sama pentingnya dengan isi informasi.

WorkspaceGraph harus mampu membangun, memperbarui, dan memvisualisasikan
hubungan tersebut.

------------------------------------------------------------------------

# Workspace Vision

Workspace terdiri dari:

-   Knowledge
-   Projects
-   Tasks
-   Templates
-   SOP
-   Daily Notes
-   Journal
-   Rules
-   References
-   Documents
-   AI Conversations

Semuanya berada dalam satu Vault.

------------------------------------------------------------------------

# Design Principles

WorkspaceGraph harus selalu:

-   sederhana
-   cepat
-   ringan
-   dapat digunakan offline
-   mudah dipelajari
-   mudah dicadangkan
-   mudah dipindahkan
-   mudah diperluas

------------------------------------------------------------------------

# Non Goals

WorkspaceGraph bukan:

-   media sosial
-   cloud storage
-   aplikasi chat
-   pengganti IDE
-   pengganti Microsoft Office
-   pengganti Obsidian

WorkspaceGraph adalah AI Workspace.

------------------------------------------------------------------------

# Success Definition

WorkspaceGraph dianggap berhasil apabila:

Pengguna dapat membuka satu aplikasi dan:

-   memahami seluruh pengetahuannya,
-   menemukan informasi dalam hitungan detik,
-   menyelesaikan pekerjaan bersama AI,
-   melihat hubungan antar ide,
-   dan membangun knowledge yang terus berkembang selama bertahun-tahun.

------------------------------------------------------------------------

# Final Statement

WorkspaceGraph dibangun bukan untuk membuat AI lebih pintar.

WorkspaceGraph dibangun untuk membuat manusia memiliki ruang kerja
digital yang terorganisasi, dapat dipahami AI, dan tetap sepenuhnya
berada dalam kendali pemiliknya.

AI akan datang dan pergi.

Model akan terus berubah.

Tetapi pengetahuan pengguna harus tetap bertahan.

WorkspaceGraph menjadikan Markdown sebagai fondasi, Graph sebagai
struktur, dan AI sebagai pekerja yang membantu mengembangkan keduanya.
